📋 解决方案指南

  1. 创建新的Personal Access Token:
    访问 https://github.com/settings/tokens
  2. 选择正确的权限:
    必须勾选完整的 repo 权限（包括所有子权限）
  3. 如果是组织仓库:
    需要组织管理员授权该Token访问组织仓库
  4. 检查仓库可见性:
    确保你有访问该仓库的权限
