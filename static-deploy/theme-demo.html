🎨 主题调色盘功能演示

🌈 丰富的主题选择

提供12种精美的预设主题，包括经典黑夜、海洋蓝、日落橙、森林绿等多种风格，满足不同用户的审美需求。

🎯 自定义渐变

支持自定义颜色选择器，用户可以选择任意两种颜色创建独特的渐变背景，打造个性化的编辑环境。

💾 主题记忆

自动保存用户选择的主题设置，下次打开应用时会自动应用上次使用的主题，提供一致的用户体验。

📱 响应式设计

主题选择器在移动端和桌面端都有优化的显示效果，确保在各种设备上都能流畅使用。

✨ 使用方法

1. 点击编辑器顶部的彩色调色盘按钮 🎨

2. 在弹出的主题选择器中选择喜欢的预设主题

3. 或者使用自定义颜色选择器创建独特的渐变

4. 主题会立即应用并自动保存
